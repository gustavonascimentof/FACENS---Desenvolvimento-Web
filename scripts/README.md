# 📋 Scripts Auxiliares

## Descrição

Scripts úteis para desenvolvimento e deploy do projeto.

## Scripts Disponíveis

### Development

```bash
npm run dev        # Inicia frontend (Vite) em desenvolvimento
```

### Build

```bash
npm run build      # Build do frontend para produção
npm start          # Inicia backend em produção
```

### Linting

```bash
npm run lint       # Executa ESLint no projeto
```

### Preview

```bash
npm run preview    # Preview do build frontend
```

## Convenções

- Use `npm run dev` para iniciar ambos os servidores
- Verifique o `.env` antes de executar
- Limpe cache com `npm run clean` se necessário
